# 项目Background Recovery说明

## 基本信息

**作者**：杨诺萁

**修改日期**：21/10/4

**简介**：包含类**Recovery**的定义和实现，实现从给定的视频中读取帧，按**累积帧**和**累积绝对值帧间差分**后取平均的方法建立背景模型，从而提取背景。

***

## 类说明

### 方法与接口

**读取视频**：

```C++
void VideoCapture(string FilePath)
```

**处理帧与背景建模**：

```C++
void Process()
```

**展示背景结果**

```C++
void show()
```

### 编译

```shell
# 先切换到工程目录下
make clearall
make
```

### 运行

```shell
./bin/test1 ./resource/video0.mp4 #示例
```
